SF2 — Перевірка формату файлу

Що означає:
Файл не відповідає переліку дозволених форматів для обраної структури даних. Наприклад, для обмінного файлу Реєстру адрес дозволено .shp та .geojson, тоді як для містобудівної документації — .gbd, .gpkg, .geojson.

Чому це важливо:
Невідповідний формат унеможливлює завантаження файлу до відповідного реєстру

Як виправити:
Експортуйте або конвертуйте вихідний файл у один із форматів, що вимагаються(через QGIS або інші ГІС системи).
Переконайтеся, що файл після конвертації відкривається у QGIS без попереджень про відсутність драйвера.

Алгоритм перевірки:
1. Визначається тип структури (регламентується параметрами валідації або метаданими).
2. Формується список дозволених розширень для цього типу.
3. Зіставляється фактичне розширення файлу з переліком.
4. Якщо формат не входить до списку — генерується помилка SF2
